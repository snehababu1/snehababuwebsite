# Sneha Babu Portfolio — Project Memory

## What this project is
Personal portfolio website for Sneha Babu, hosted on GitHub Pages at snehababu.eu
Built using HTML, CSS, vanilla JavaScript only. Two files: index.html and blog.html

## Design decisions locked
- Accent color: coral-pink #E8506A
- Background: warm cream #FAFAF7
- Font: Poppins (Google Fonts)
- SB monogram logo in navbar (not full name)
- EN/DE language toggle in navbar
- No external frameworks except particles.js (hero background)

## Sections in index.html (in order)
1. Hero — blob animation, doodles, typewriter roles, Get in Touch + Request CV buttons
2. About Me — circular photo (sneha.jpeg), flags row, hobbies bubbles, education badges
3. Skills — Currently Learning first (pulse dot), then Marketing, Project Management, Interpersonal, Design Tools, Languages
4. Experience — tabbed layout (left tabs, right content panel)
5. Projects — flip cards: Hues of Hamburg (stealth), Freelance Client (anonymous, was AronLabz), Independent Publishing (no KDP mention)
6. Certifications — real SVG logos, badge wall
7. Blog — teaser section with button linking to blog.html
8. Contact — email form + CV request modal popup

## blog.html
- Separate page with same navbar and footer
- 6 blog cards, filterable by category (All, Personal, Career, Marketing & AI)
- Blog posts not yet written in full — placeholders only
